Git 的使用 并传送文件至github

注册github帐号

github.com 去注册一个自己的帐号

安装Git

yay -S git

初始化配置GIT

配置用户信息

git config --global user.name "your name"
git config --global user.email "your email"

上传文件

进入要传送的文件夹内，

初始化本地仓库

git init

添加拟上传的对象

//上传文件夹1
git add f1

//上传指定文件
git add *.*

//上传所有修改的文件
git add .
or
Git add -A

将修改后的文件提交到本地仓库

git commit -m ‘增加README.md说明文档’

连接到远程仓库

git remote add origin 远程仓库地址

推送

git push -u origin master
